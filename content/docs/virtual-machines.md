## Быстрый старт

Откройте Dashboard, выберите проект и нажмите **Создать VM**. Укажите регион, образ и конфигурацию сервера.

## Выбор конфигурации

Для веб-приложений начните с универсальной серии. Для баз данных и сборки проектов выбирайте конфигурации с большим объёмом памяти или CPU.

| Серия | Назначение | Стартовая конфигурация |
| --- | --- | --- |
| General | API и веб-приложения | 2 vCPU / 4 GB |
| Compute | Сборка и обработка | 4 vCPU / 8 GB |
| Memory | Базы данных | 4 vCPU / 16 GB |

## Снимки и резервные копии

Создавайте снимок перед крупным обновлением. Снимки можно использовать для быстрого клонирования серверов.
